# Food Trends Understanding Customer Preferences in FB
## Project Overview
This project presents a comprehensive analysis of nutrition, dietary behavior, and lifestyle patterns through a set of interactive Power BI dashboards. It explores how daily food choices, meal distribution, physical activity, and health conditions collectively shape nutritional intake and overall well-being. The dashboards are built to deliver clear, data-driven insights through dynamic visualizations and filters, allowing users to interact with the data, identify trends, and better understand the relationship between lifestyle decisions and health outcomes.
## Objectives
- Analyze dietary patterns across different meal types
- Evaluate nutritional intake (calories, protein, fat, carbohydrates, sugar)
- Understand the impact of exercise levels and lifestyle habits
- Study relationships between diet and health conditions (BMI, diseases)
- Compare diet types (Vegetarian, Vegan, Non-Vegetarian)
- Provide data-driven insights for healthier lifestyle decisions
